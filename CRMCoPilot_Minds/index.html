Overview

This documentation provides detailed information about the Mind workflows implemented for a Series A funded supply chain SaaS company's CRM Copilot system. Minds are specialized AI components that process various data sources to provide valuable insights and automations.

Two primary categories of Minds are documented:

  • Raw Data Collection Minds: These extract, process, and vectorize data from various sources like Salesforce, Email, Slack, Zoom, etc.
  • AI-Processed Collection Minds: These analyze the vectorized data to generate insights, profiles, and recommendations.

Collections

The following collections (vector databases) are used to store and retrieve processed data. They are divided into Raw Collections (directly populated from data sources) and AI-Processed Collections (generated through analysis of raw data).

Raw Collections

Collection Name Description Data Source Populated By
sales_activities Sales opportunities, proposals, and account information Salesforce, Document storage SalesforceOpportunityIngestor, SalesforceAccountProfiler, QuoteProposalExtractor
customer_interactions Customer emails, support tickets, and other direct communications Email servers, Support platforms EmailCommunicationProcessor, SupportTicketProcessor
internal_discussions Internal team communications about customers Slack, Teams SlackThreadExtractor
meeting_records Customer meeting transcripts and summaries Zoom, Teams ZoomTranscriptVectorizer, TeamsCallAnalyzer

AI-Processed Collections

Collection Name Description Source Collections Generated By
customer_profiles AI-generated comprehensive customer profiles sales_activities, customer_interactions CustomerProfileGenerator, ExecutiveSummaryCreator
relationship_timelines Customer journey maps and interaction histories All raw collections CustomerJourneyMapper
opportunity_insights Sales opportunity analysis and recommendations sales_activities, meeting_records, customer_profiles OpportunityAnalyzer, UpsellOpportunityDetector, QuarterlyBusinessReviewer
risk_assessments Customer churn risk evaluations All raw collections ChurnRiskCalculator
sentiment_analysis Customer sentiment trends over time customer_interactions, meeting_records SentimentTrendAnalyzer

Raw Data Collection Minds

These Minds extract and process data from various sources, then embed it into vector collections for further use.

EmailCommunicationProcessor

Source: Company Email Server

Output: customer_interactions

Purpose: Process customer emails with metadata

SlackThreadExtractor

Source: Slack API

Output: internal_discussions

Purpose: Extract customer-related Slack discussions

ZoomTranscriptVectorizer

Source: Zoom API

Output: meeting_records

Purpose: Process meeting transcripts and metadata

SupportTicketProcessor

Source: Service Desk API

Output: customer_interactions

Purpose: Process support tickets with priority and status

QuoteProposalExtractor

Source: Document Storage

Output: sales_activities

Purpose: Extract data from sales proposals

SalesforceOpportunityIngestor

Source: Salesforce API

Output: sales_activities

Purpose: Extract and vectorize opportunity data

SalesforceAccountProfiler

Source: Salesforce API

Output: sales_activities

Purpose: Extract customer account information

TeamsCallAnalyzer

Source: Teams API

Output: meeting_records

Purpose: Extract insights from Teams calls

AI-Processed Collection Minds

These Minds analyze vectorized data from raw collections to generate insights, profiles, and recommendations.

CustomerProfileGenerator

Sources: sales_activities, customer_interactions

Output: customer_profiles

Purpose: Create comprehensive customer profiles

CustomerJourneyMapper

Sources: All raw collections

Output: relationship_timelines

Purpose: Generate timeline of customer interactions

OpportunityAnalyzer

Sources: sales_activities, meeting_records

Output: opportunity_insights

Purpose: Analyze sales opportunities with win probability

ChurnRiskCalculator

Sources: customer_interactions, sales_activities

Output: risk_assessments

Purpose: Identify churn risk indicators

SentimentTrendAnalyzer

Sources: customer_interactions

Output: sentiment_analysis

Purpose: Track customer sentiment over time

UpsellOpportunityDetector

Sources: customer_profiles, sales_activities

Output: opportunity_insights

Purpose: Identify potential upsell opportunities

QuarterlyBusinessReviewer

Sources: customer_profiles, opportunity_insights

Output: opportunity_insights

Purpose: Generate QBR materials automatically

ExecutiveSummaryCreator

Sources: customer_profiles, risk_assessments

Output: customer_profiles

Purpose: Create executive-level customer summaries
